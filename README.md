# Resas API x React

## 環境変数

Resas API を発行して .env に貼り付け

`REACT_APP_RESAS_API_KEY="sampleAPIKey"`

## Available Scripts

In the project directory, you can run:

### `yarn start`

install Some librarys that need in this project.

### `yarn install`

ESlint

### `yarn lint`

### `yarn lint:fix`

prettier

### `yarn prettier`

### `yarn prettier:fix`
